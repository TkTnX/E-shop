# E-shop - интернет-магазин электроники (MERN, FSD)

## Мой первый проект, где я пытаюсь использовать методологию Feature-Sliced Design. Также попробовал новый state manager - MobX.

### Использованные технологии:

Frontend (MERN + FSD):

- React
- Mobx
- SCSS
- TypeScript
- React-Router
- React-Query

Backend:

- Node.js
- Express
- MongoDB
- Mongoose
